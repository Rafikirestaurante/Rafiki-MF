# Resumen — Rafiki MF Fase 3A.2

Versión **1.3.3**. Sobre la base 1.3.2 se agregó una nueva regla al extractor de Bancolombia para reconocer alertas del tipo `Recibiste un pago [TIPO] de [ORIGEN] por $[VALOR]`. El caso real incorporado es `Recibiste un pago PROVEEDOR de REDEBAN SA por $114109.00 en tu cuenta de Ahorros el 17/07/2026 a las 17:13`. Rafiki MF registra este mensaje como ingreso Bancolombia por 114109 COP, detalle `REDEBAN SA`, fecha 17/07/2026 y hora 17:13; además conserva `PROVEEDOR`, el origen y el tipo de cuenta dentro de `source_metadata`. Se mejoró la extracción horaria para priorizar expresiones explícitas como `a las 17:13` sobre marcas horarias visuales que puedan aparecer antes en el texto del correo. La forma anterior `Recibiste un pago de ...` sigue soportada. No se añadieron migraciones SQL ni dependencias nuevas. La Edge Function `gmail-sync-now` debe redesplegarse porque utiliza el extractor compartido actualizado.
